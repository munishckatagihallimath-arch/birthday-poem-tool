# 🎂 Birthday Poem Generator

A free web tool that generates beautiful birthday poems in any language using AI.

## Features
- 16 languages — Hindi, English, Marathi, Bengali, Tamil, Punjabi and more
- 7 poem styles — Heartfelt, Funny, Romantic, Inspirational and more
- Typewriter animation effect
- Copy and Share buttons
- Works on mobile and desktop

## Languages Supported
Hindi, English, Marathi, Bengali, Tamil, Punjabi, Kannada, Gujarati, Malayalam, Telugu, Urdu, Spanish, French, Arabic, Japanese, Russian

## How to Use
1. Enter the person's name
2. Choose a language
3. Choose a poem style
4. Click Generate — enjoy your poem!
